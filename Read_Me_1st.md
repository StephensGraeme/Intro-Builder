*** Intro Builder ***
Just a basic little vibe coded (Avec Monsieur Claude Esq.) utility written in python to create custom pre-roll intro video's for the use in Jellyfin, Plex etc.

*** Ensure you have set 'execute' permissions on the file (Read the manual) ***

A neat little trick: All setting's can be stored in the output file as metadata so the look and feel can be easily transplanted from a previously generated intro (Use the big button at the top).

Prerequisits: Python3 & the latest versions of ffmpeg (inc. ffprobe) & VLC.

It's pretty much self explanitory, basically work your way through the tabs, then press the build button.
It will let you know if you have missed something.

Tested on a RPi 5 8GB & Manjaro 12GB (please let me know if it works or not on other systems)

Enjoy!
